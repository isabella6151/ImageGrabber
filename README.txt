ComplianceImageGrabber
-----------------------

Purpose:
To capture snapshots of views from all identified cameras.


How to run:
1. The included opts file identifies all the information the script requires to run. It needs to be updated with the relevant information.
2. create a scheduled task that runs the batch file run.bat. This should trigger on the hour every hour.
3. Make sure you create appropriate folders in the FTP area for the account you are using to ftp images back.
